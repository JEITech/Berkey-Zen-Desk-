Viewing intent:

Utterances

Utterances are the trigger phrases that the bot will pick up on to determine a customers intent. These do not have to be exact matches, as artificial intelligence is used to pair similar statements to existing utterances. These work best when they are no longer than 2-3 words. You can edit/delete an utterance by clicking on it, or add a new one by clicking "Add utterance" below.

Responses

A response is made up of 5 message groups. Each message group can contain up to 5 messages. One message from each group is chosen at random for each response. A message group will only be saved if there is a message in that group. You can add, remove, and edit message groups/messages below.
